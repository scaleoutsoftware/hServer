This sample illustrates the usage model of passing an object as a parameter to all nodes in a SOSS store.

To compile the sample:
javac -classpath "/usr/local/soss/java_api/lib/*:/usr/local/soss/java_api/hslib/hadoop-1.2.1/*:/usr/local/soss/java_api/*" com/scaleoutsoftware/soss/hserver/examples/ParameterPassing.java

To Run the sample:
java -classpath "/usr/local/soss/java_api/lib/*:/usr/local/soss/java_api/hslib/hadoop-1.2.1/*:/usr/local/soss/java_api/*:./*" com.scaleoutsoftware.soss.hserver.examples.ParameterPassing intMin intMax
